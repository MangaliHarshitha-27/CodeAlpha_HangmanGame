
# Hangman Game - CodeAlpha Internship

This is a simple text-based Hangman Game built using Python.

## Features
- Random word selection
- 6 incorrect attempts allowed
- Input validation
- Win/Lose detection

## Concepts Used
- Random module
- While loop
- If-else
- Lists and strings
- Input/Output

## How to Run
1. Install Python
2. Run: python hangman.py
